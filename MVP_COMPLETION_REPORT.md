
## 🔧 本番運用の準備

### **必要な設定**
1. **API キー取得**:
   - Google Places API キー
   - 楽天アプリケーション ID
   
2. **環境変数設定**:
   ```bash
   GOOGLE_PLACES_API_KEY=your_google_api_key
   RAKUTEN_APP_ID=your_rakuten_app_id
   ```

3. **サーバー起動**:
   ```bash
   python -m uvicorn api.main:app --host 0.0.0.0 --port 8000
   ```
